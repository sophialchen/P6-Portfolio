Mobile Music Player Representation

Using HTML, CSS, and JavaScript, I coded an interface for a mobile music player.

I did this project for my Human-Computer Interaction class at Northwestern. The course was my first introduction to coding with HTML, CSS, and JavaScript.

Album-select page

On the album-select page, users can select the album they want to listen to

Song-select page

On the song-select page, users can select the song they want to listen to

Song player page

On the song page, users can play and pause their song. The player keeps track of the time elapsed. Users can also skip to the next or previous songs, as well as change the volume.
